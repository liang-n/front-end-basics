14-文本样式

字体样式针对的是“文字本身”的形体效果，而文本样式针对的是“整个段落”的排版效果，因此在CSS中，特意使用了“font”和“text”两个前缀来区分这两类样式。

在CSS中，常见的文本样式如下：

  • text-indent - 首行缩进
  • text-align - 水平对齐
  • text-decoration - 文本修饰
  • text-transform - 大小写转换
  • line-height - 行高
  • letter-spacing/word-spacing - 文字间距/单词间距

text-indent 首行缩进

中文段落首行一般需要缩进两个字的空间，想要实现这个效果，那么textindent值应该是font-size值的2倍。

text-align 水平对齐

text-align属性取值有 3 个：

  • left - 左对齐（默认）
  • center - 居中
  • right - 右对齐

text-decoration 文本修饰

使用 text-decoration 属性来定义文本的修饰效果：

  • none - 去除所有所有划线效果（默认）
  • overline - 上划线
  • line-through - 中划线（删除线）
  • underline - 下划线

text-transform 大小写转换

text-transform属性是针对英文而言的，因为中文不存在大小写之分。

text-transform属性取值有4个：

  • none - 无转换（默认）
  • uppercase - 转换为大写
  • lowercase - 转换为小写
  • capitalize - 首字母大写

letter/word-spacing 字符／单词间距

letter-spacing，从英文意思上就可以知道这是“字母间距”。注意，每一个中文汉字都被当作一个“字”，而每一个英文字母也被当作一个“字”。

word-spacing，从英文意思上就可以知道这是“单词间距”。一般来说，word-spacing只针对英文单词而言。
