09-表单

在HTML中，表单标签有五种：form、input、textarea、select和option。

从外观上来划分，表单可以分为以下八种：

  1. 单行文本框
  2. 多行文本框
  3. 密码文本框
  4. 单选框
  5. 复选框
  6. 下拉列表
  7. 按钮
  8. 文件上传

form 标签

创建一个表单，就像创建一个表格，我们也必须要把所有表单标签放在form标签内部。

name 属性

在一个页面中，表单可能不止一个，每一个form标签就是一个表单。为了区分这些表单，我们可以使用name属性来给表单进行命名。

method 属性

在form标签中，method属性用于指定表单数据使用哪一种http提交方法。

method属性取值有两个:

  • get
  • set

action 属性

在form标签中，action属性用于指定表单数据提交到哪一个地址进行处理。

target 属性

在form标签中，target属性与a标签的target属性是一样的，都是用来指定窗口的打开方式。

一般情况下，我们只会用到_blank这一个属性值。

enctype 属性

在form标签中，enctype属性用于指定表单数据提交的编码方式。

一般情况下，我们不需要设置，除非你用到上传文件功能。

input 标签

在HTML中，大多数表单都是使用input标签来实现的。

使用属性 type 控制显示的表单类型，type 值如下：

  • text - 单行文本框
  • password - 密码输入框
  • radio - 单选框
  • checkbox - 多选框
  • button/submit/reset - 按钮
  • file - 文件上传

text 单行文本框

text 常用属性，使用时不限制先后顺序：

  • value - 设置文本框默认值
  • size - 设置文本框长度
  • maxlength - 设置文本框最大输入字符数量

password 密码输入框

密码文本框在外观上与单行文本框相似，两者也拥有相同的属性：value、size、maxlength等。

不过两者也有着本质上的区别：在单行文本框中输入的字符可见，而在密码文本框中输入的字符不可见。

radio 单选框

radio 有两个必须设置的属性：

  • name - 表示选项所在的组
  • value - 表示选项的值

非必须的属性：

  • checked - 设置默认选项
性别： 男 女

一个表单可能会有多组单选框，不同组的 name 属性必须设置不同的值，同组的单选框，必须设置一样的 name 属性值。

checkbox 多（复）选框

单选框只能选择一项，而复选框可以选择多项，同时与单选框一样，name 与 value 是必须设置的属性。

可选属性 checked 也是设置默认选中的选项，在这里可以同时勾选多个。

按钮

button 按钮

普通按钮一般情况下都是配合JavaScript来进行各种操作的。

submit 按钮

提交按钮一般用来给服务器提交数据。我们可以把提交按钮看成是一种特殊功能的普通按钮。

reset 按钮

重置按钮一般用来将 form 标签之内的所有 input 标签重置到初始化状态，对所处 form 标签外不生效，submit 提交按钮同理。

textarea 多行文本框

单行文本框只能输入一行文本，而多行文本框却可以输入多行文本。

在HTML中，多行文本框使用的是textarea标签，而不是input标签。多行文本框的默认显示文本是在标签对内部设置，而不是在value属性中设置的。

除了与单行输入框相同的属性，还可以使用 rows 和 cols 设置高度和宽度。
