18.3 图片对齐


18.3.1 水平对齐

在 img 元素的父级元素中使用 text-align 属性来定义内容图片的水平对齐方式。

语法：text-align: 取值;

  • left - 左对齐（默认）
  • center - 居中对齐
  • right - 右对齐

text-align 属性一般只用于两个地方：文本水平对齐和图片水平对齐。

示例：




18.3.2 垂直对齐

在 img 元素上使用 vertical-align 属性来定义图片周围的行内元素或文本相对与该元素垂直对齐。

语法：vertical-align: 取值;

  • top - 顶部对齐
  • middle - 中部对齐
  • baseline - 基线对齐
  • bottom - 底部对齐

vertical 是「垂直」的意思，align 是「排列」，学习 CSS 属性与学习 HTML 标签一样，根据英文意思去理解和记忆事半功倍。

可以观察到 baseline 和 bottom 的是不一样的：baseline 会将图片与文字内容的底部对齐，bottom 则将图片与文字行高对齐。


18.4 文字环绕 float

网页布局中，经常进行图文混排，使用 float 属性实现文字环绕图片的效果。

语法：float: 取值;

  • left - 元素向左浮动
  • right - 元素向右浮动
